Então, decidi escrever este artigo como uma forma de backup do que aprendi até então e também para compartilhar o conhecimento com quem tiver interesse.

Neste artigo, não tentarei explicar nada sobre a história da contabilidade ou sua utilidade. Para tal, já existem diversos excelente materiais de estudo espalhados pela internet. Introdução

Empresas lidam com dinheiro. Cada país utiliza uma moeda diferente. Por isso precisamos encontrar uma forma de representar quantidades de dinheiro programaticamente. Para tal, utilizamos o padrão ISO 4217.

Podemos criar uma classe imutável para representar um valor em dinheiro da seguinte forma:


// Typescript

class Dinheiro {
    private _valorEmCentavos: number;
    private _moeda: string;

    constructor(valorEmCentavos: number, moeda: string) {
        this._valorEmCentavos = Math.ceil(valorEmCentavos);
        this._moeda = moeda;
    }

    get valorEmCentavos() {
        return this._valorEmCentavos;
    }

    get moeda() {
        return this._moeda;
    }
}

// exemplo
const cemReais = new Dinheiro(100_00, "BRL");

Para manter as coisas simples, por enquanto utilizaremos somente o Real Brasileiro, que no padrão ISO 4217 é chamado de BRL.

Ativos e Passivos

Toda empresa é dona de algumas coisas e deve outras coisas. Por exemplo, uma empresa pode ser dona de carros, de imóveis, do dinheiro que está no caixa, de marcas e patentes, etc., e também pode ter dívidas a pagar, salários a depositar, lucros a distribuir, etc.

As coisas das quais a empresa é dona são chamadas de "Ativos", e as coisas que a empresa deve são chamadas de "Passivos".


// Typescript

type CoisaComValorMonetario = {
    nome: string;
    valor: Dinheiro;
};

type Ativo = CoisaComValorMonetario;

type Passivo = CoisaComValorMonetario;

Os Ativos e Passivos podem ser:

Circulantes ou Não Circulantes
Coisas que vão gerar alguma movimentação de valores dentro de um prazo menor que 1 ano, são chamadas "Circulantes" porque tem impacto no curto prazo. O contrário é chamado de "Não Circulante", por não ter impacto no curto prazo. Exemplos de Ativos Circulantes são o dinheiro em caixa e o estoque de mercadorias. A Equação Fundamental da Contabilidade

Existem 3 termos que precisamos definir anter de ir em frente: "Ativo", "Passivo", e "Patrimônio Líquido"

Ativo
Tudo o que a empresa possui e tem o direito de receber. Exemplo: Dinheiro em espécie, valores a receber, carros, casas, etc.
Passivo
Basicamente, uma forma chique de dizer "dívida".
Patrimônio Líquido
O quanto a empresa deve para os donos e/ou acionistas, que representa exatamente a diferença entre o Ativo e o Passivo.

Agora que definimos o que estes termos significam, podemos conhecer a Equação Fundamental da Contabilidade:

Ativo = Passivo + Patrimônio Líquido

Algumas vezes podemos encontrar essa mesma conta um pouco diferente, mas que na verdade significa a mesma coisa: "Patrimônio Líquido = Ativo - Passivo", o que faz sentido, porque basicamente signica que o patrimônio de uma empresa é igual a tudo o que ela tem menos tudo o que ela deve.

Podemos expressar isso com um pouco de código:


// Typescript

class Empresa {
    constructor(
        private bens_e_direitos: CoisaComValorMonetario[],
        private dividas_e_obrigacoes: CoisaComValorMonetario[],
    ) { }

    get total_ativos() {
        return this.bens_e_direitos.reduce((total, bem) => {
            return total + bem.valor;
        }, 0);
    }

    get total_passivos_exceto_patrimonio_liquido() {
        return this.dividas_e_obrigacoes.reduce((total, obrigacao) => {
            return total + obrigacao.valor;
        }, 0);
    }

    get patrimonio_liquido() {
        return this.ativo - this.passivo;
    }
}
